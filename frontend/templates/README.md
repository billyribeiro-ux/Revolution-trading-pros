# Templates Directory

This directory contains extracted text content from Word document templates used in the Revolution Trading platform.

## Files

| File | Description | Original Format |
|------|-------------|-----------------|
| `SimplerMain.txt` | Main WordPress/Beaver Builder template with JavaScript functionality | `.dotx` |
| `SimplerSmall.txt` | Articulate Storyline course embed - "Quickstart to Precision Trading" | `.docx` |
| `SimplerThankYou.txt` | Thank you page template | `.dotx` |
| `WordpressTheme.txt` | WordPress theme configuration and scripts | `.docx` |

## Purpose

These templates contain:
- WordPress/Beaver Builder layout configurations
- JavaScript functionality for the trading platform
- Articulate Storyline course embeds
- Dashboard customizations and UI components

## Usage

These files are plain text extractions from Microsoft Word documents, making them readable by:
- AI agents and language models
- Code search tools
- Version control systems
- Automated documentation generators
