Creating a Countdown Popup

{ countdownTimer: { enabled: true, endDate: '2025-12-31T23:59:59Z', showDays: true, showHours: true, showMinutes: true, showSeconds: true, timerColor: '#6366f1', onExpire: 'hide' } }

Embedding a Video

{ videoEmbed: { enabled: true, url: 'https://www.youtube.com/watch?v=VIDEO_ID', autoplay: false, muted: false, controls: true, aspectRatio: '16:9' } }

Advanced Targeting
